## Steps to Replace Images

1. Navigate to `/frontend/public/assets/` in your project folder
2. Replace each image file with your own, keeping the exact filename
3. Ensure images are in common formats (jpg, png, pdf)
4. No code changes needed - just replace and save!
5. Refresh your browser to see the new images

## Image Recommendations

- **Profile Image**: 400x400 pixels, professional headshot
- **Hero Image**: 1200x800 pixels, high quality
- **Project Images**: 800x600 pixels minimum for best quality
- **Testimonial Images**: 200x200 pixels, circular portraits
- **All Images**: Use JPG for photos, PNG for graphics with transparency

## Tips

- Keep file names exactly as listed (they're case-sensitive on some systems)
- Optimize images for web (compress if possible)
- Use consistent image quality across all images
- For testimonials, ensure photos are consistent in style and quality
